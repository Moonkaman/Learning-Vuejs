# quiz

This project was made using the Vue CLI to generate boilerplate, Bootstrap Vue, lodash and is pulling quiz data from https://opentdb.com/

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
